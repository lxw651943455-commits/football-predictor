# 足球预测系统 - 分享给朋友使用指南

## 🎯 方案概述

使用 **Cloudflare Tunnel** 免费分享，不占用本地端口，随时可以关闭。

---

## 📦 方案一：最简单（推荐）

### 使用 expose 或 localhost.run

```bash
# 安装 expose
npm install -g expose

# 启动分享（前端+后端）
cd ~/football-predictor-app
expose 3001 5000 --subdomain=你的名字-football
```

**你会得到一个公网地址**：
```
https://你的名字-football.expose.run
```

直接把这个链接发给朋友，他们就能访问了！

**优点**：
- ✅ 超级简单，一条命令搞定
- ✅ 自动生成 HTTPS
- ✅ 不需要配置
- ✅ 关闭终端就停止分享

**缺点**：
- ⚠️ 需要保持终端运行
- ⚠️ 免费版可能限速

---

## 📦 方案二：Cloudflare Tunnel（稳定）

### 1. 安装 cloudflared

```bash
# Windows (下载)
# https://github.com/cloudflare/cloudflared/releases/download/2024.3.0/cloudflared-windows-amd64.exe

# 放到 C:\Program Files\cloudflared\
# 添加到 PATH
```

### 2. 创建分享配置

创建文件 `share-tunnel.bat`：

```batch
@echo off
echo 正在启动公网分享...
echo.

REM 分享前端（端口 3001）
start "Frontend Tunnel" cloudflared tunnel --url http://localhost:3001

REM 分享后端（端口 5000）
start "Backend Tunnel" cloudflared tunnel --url http://localhost:5000

echo.
echo ✅ 公网分享已启动！
echo.
echo 请查看弹出的窗口，会显示公网地址
echo 把 https://开头的地址发给朋友
echo.
echo 关闭此窗口会停止分享
echo.
pause
```

### 3. 启动分享

双击 `share-tunnel.bat`，会弹出两个窗口显示公网地址。

### 4. 停止分享

关闭所有弹出的窗口即可。

## 📦 方案三：Vite 内置（开发用）

修改 `frontend/vite.config.js`，已配置好：

```javascript
server: {
  host: '0.0.0.0', // 允许局域网访问
}
```

**局域网分享**：
1. 查看你的本机 IP：
```bash
ipconfig
# 找到 "IPv4 地址"，例如 192.168.1.100
```

2. 告诉朋友访问：
```
http://192.168.1.100:3001
```

**注意**：朋友需要能连接到你的 Wi-Fi 或网络。

## ✅ 使用步骤（推荐方案一）

1. **安装工具**（只需一次）：
```bash
npm install -g expose
```

2. **启动服务**：
```bash
cd ~/football-predictor-app
# 确保所有服务都在运行
```

3. **启动分享**：
```bash
expose 3001 5000 --subdomain=my-football
```

4. **复制链接**：
终端会显示类似：
```
https://my-football.expose.run
```

5. **发给朋友**，他们就能直接访问了！

6. **停止分享**：按 `Ctrl+C` 关闭终端

---

## ⚠️ 注意事项

1. **分享时不要关闭服务**
   - 前端、后端、Python 引擎都需要保持运行

2. **不要暴露敏感信息**
   - API Keys 已配置在服务端，朋友无法看到

3. **性能考虑**
   - 免费隧道可能有限速
   - 建议同时使用的人数 < 10

4. **随时关闭**
   - 关闭分享终端即可停止公网访问
   - 不影响本地使用

---

## 🔒 安全提示

- ✅ API Keys 在服务端，朋友无法访问
- ✅ 只读访问，朋友不能修改数据
- ✅ 随时可以关闭公网访问
- ❌ 不要分享给不信任的人

---

## 🆘 遇到问题？

1. **朋友打不开**：
   - 检查服务是否运行
   - 检查分享终端是否还在运行
   - 尝试刷新页面

2. **连接超时**：
   - 免费隧道可能不稳定
   - 建议使用 Cloudflare Tunnel

3. **想停止分享**：
   - 直接关闭分享终端
   - 不影响本地使用
