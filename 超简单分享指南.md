# 超简单分享方案

## 🎯 推荐方案：localtunnel

### 一行命令搞定：

```bash
npx localtunnel --port 3001 --subdomain my-football
```

**就这么简单！**

会生成一个网址，例如：
```
https://my-football-3001.localtunnel.me
```

**发给朋友这个地址就行了！**

---

## 📦 或使用 ngrok

### 步骤：

1. **注册**：访问 https://ngrok.com/signup（免费）

2. **下载**：下载 Windows 版本

3. **解压**：解压到任意文件夹

4. **运行**：
```bash
ngrok http 3001
```

5. **复制地址**：在命令行窗口中找到 `https://xxxx.ngrok-free.app`

## ⚡ 最简单的方法

**直接运行这个批处理文件**：
```
最简单的分享.bat
```

选择方案 1，自动运行 localtunnel！

---

## ✅ 优点

- ✅ 无需安装（使用 npx）
- ✅ 自动 HTTPS
- ✅ 一行命令
- ✅ 关闭窗口就停止分享

---

## 🔍 如何找到公网地址？

运行后会显示：
```
your url is: https://my-football-3001.localtunnel.me
```

复制这个 `https://` 开头的地址给朋友！
